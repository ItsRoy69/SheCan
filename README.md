<h1 align="center">
Getting started
</h1>

<p align="center">
First, make sure you have all these installed on your local machine & then continue.

- Node.js v16.13.1^
- NPM v8.1.0^

### Installation

```bash
# Clone the repository
git clone
npm install --force

# Enter into server
cd server
npm install
```

### Starting the application

```bash
# To start the client application
npm run dev
# To start server application
npm start
```
